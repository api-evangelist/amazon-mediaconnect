# Amazon MediaConnect (amazon-mediaconnect)
AWS Elemental MediaConnect is a high-quality transport service for live video that provides the reliability, security, and visibility customers expect from traditional satellite and fiber services. It enables broadcasters to build live video workflows in the cloud with reliable transport of broadcast-quality content.

### AWS Elemental MediaConnect API
